# Прямая линия с Главой Республики Башкортостан

Статический лендинг для анонса прямой линии и приёма вопросов через социальные сети.

## Замена медиафайлов

- **`assets/img/hero.jpg`** — фон геро-блока. Замените файлом подходящего размера (рекомендуется не менее 1920×1080). Имя файла и расширение должны сохраниться.
- **`assets/video/hero.mp4`** — видеозаставка для фонового блока. Подготовьте короткий MP4 (5–10 секунд) с соотношением сторон 16:9 и поместите его в эту же папку, оставив имя файла `hero.mp4`.

После замены файлов очистите кэш браузера или сделайте жёсткое обновление страницы для отображения обновлений.

## Ссылки на каналы обратной связи

В блоке «Подать обращение через социальные сети» (файл `index.html`) замените заглушки `#` на фактические ссылки приёма обращений:

- ВКонтакте — `channels--vk`
- Одноклассники — `channels--ok`
- Telegram — `channels--tg`
- Госуслуги. Решаем вместе — `channels--gosuslugi`

## Локальный просмотр

Откройте `index.html` в браузере. Для корректной работы фонового видео рекомендуется запускать страницу через локальный веб-сервер (например, `python -m http.server`).
