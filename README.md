# Прямая линия с Главой Республики Башкортостан

Статический промо-лендинг, анонсирующий прямую линию с главой региона и собирающий вопросы жителей через социальные сети и виджет CraftTalk. Макет выполнен на чистом HTML/CSS с минимальным количеством JS.

## Структура проекта

- `index.html` — основная разметка лендинга.
- `assets/css/app.css` — стили, дополненные под Bootstrap 5.
- `assets/js/app.js` — сценарий с анимациями и поведением геро-блока.
- `assets/js/widget.js` — оригинальный скрипт CraftTalk (оставлен для справки).
- `assets/icon/`, `assets/img/`, `assets/video/` — иконки, изображения и видеофон.

## Локальный просмотр

1. Откройте репозиторий в терминале.
2. Запустите любой лёгкий HTTP-сервер (нужно для корректной загрузки видео и виджета):
   ```bash
   python3 -m http.server
   ```
3. Перейдите в браузере по адресу `http://localhost:8000`.

CraftTalk загружается по HTTPS. При открытии страницы напрямую через `file://` внешний скрипт может быть заблокирован — используйте локальный сервер.

## Настройка контента

- **Геро-блок** — замените `assets/img/hero.jpg` и `assets/video/hero.mp4` на актуальные материалы, оставив имена файлов.
- **Социальные сети** — ссылки и подписи расположены в секции «Подать обращение через социальные сети» (`index.html`). Обновите URL при необходимости.
- **Тексты** — редактируются напрямую в `index.html`.

## Виджет CraftTalk

Внизу `index.html` размещён официальный embed-код CraftTalk для `webchat_directline_rb`. После загрузки:

- создаётся iframe с чат-виджетом;
- любые кнопки с атрибутом `data-open-chat` (например, «Задать вопрос») автоматически открывают чат по клику.

Если виджет не появляется, проверьте:

- что запросы к `https://cloud.craft-talk.ru` не блокируются адблокером/фаерволом;
- что страница открыта по HTTP/HTTPS, а не по `file://`;
- перезагрузите вкладку с очисткой кэша (`Ctrl+Shift+R` или `Cmd+Shift+R`).

## Развёртывание

Сайт статический — достаточно выложить содержимое репозитория на любой хостинг с поддержкой HTTPS (S3, GitHub Pages, Nginx и т. п.). Для обновления заменяйте соответствующие файлы и инвалидайте кэш CDN/браузера.
